创建用户

Authsaur内置密码登录认证器，可以在1分钟内创建用户。

  • 点击菜单【用户】，进入我的企业

  • 点击按钮【创建用户】，密码默认规则：8-10位，至少包含大小写字母、数字和特殊字符。

  • 用户创建成功
